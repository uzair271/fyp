# FYP Project

A professional React application with a fully responsive Navbar component.

## Features

- Fully responsive Navbar with mobile hamburger menu
- Tailwind CSS for styling
- React Router ready
- Authentication state management
- Accessibility features (ARIA labels)
- Smooth animations and transitions

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
